# ApgarTrack
Projeto Android para facilitar o cálculo do índice de Apgar.
-# Feito por Eduardo Madeira, João Vargas e João Carapuça no âmbito da Disciplina de Interação Pessoa-Computador (ESTIG, IPBeja).

Funcionalidades:
- Calculadora para vários minutos diferentes de vida (1º, 5º, 10º, 15º, ...);
- Botões e resultados claros para a experiência mais rápida possível;
- Menu de visualização de todos os testes efetuados.
