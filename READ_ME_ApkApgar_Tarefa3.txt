ApgarTrack - Instalação e utilização da APK

Ficheiro:
ApgarTrack-Cenario3-Utentes-debug.apk

Requisitos:
- Telemóvel Android ou emulador Android
- Android 7.0 ou superior
- Permissão para instalar aplicações fora da Play Store, se o sistema pedir

Instalar no telemóvel Android:
1. Transferir o ficheiro APK para o telemóvel.
2. Abrir o ficheiro ApgarTrack-Cenario3-Utentes-debug.apk.
3. Se aparecer um aviso de segurança, permitir a instalação de aplicações de origem desconhecida.
4. Confirmar a instalação.
5. Após instalar, abrir a aplicação ApgarTrack.

Instalar no emulador Android:
1. Abrir o Android Studio.
2. Iniciar um emulador Android.
3. Arrastar o ficheiro APK para a janela do emulador.
4. Aguardar a instalação.
5. Abrir a aplicação ApgarTrack.

Usar a aplicação:
1. Abrir a aplicação ApgarTrack.
2. Usar o botão Home para voltar ao ecrã inicial.
3. Usar o botão Utentes para aceder à lista de utentes.
4. Tocar num utente para ver as suas avaliações.
5. Tocar numa avaliação para ver os respetivos parâmetros.
6. Usar os botões de voltar dentro da aplicação para regressar aos ecrãs anteriores.

Nota:
Esta APK é uma versão de teste/debug e serve apenas para utilização no contexto do protótipo.
